# Branding assets

Replace these placeholders with your own files:

- **logo.png** — Main logo (e.g. 200×60 px PNG, transparent or white background for dark nav).
- **favicon.ico** — Browser tab icon (e.g. 32×32 ICO or PNG).

Without these files, the app will still run; the default Frappe/ERPNext logo and favicon may be used if the URLs 404.
